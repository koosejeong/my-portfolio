              미리미리 필요한 변수들을 선언해 줍니다 :) 

              가장 중요한 변수는 버튼의 클릭값을 담아둘 변수입니다. 

              

              

              버튼on('click', function(e){ 

                e.이벤트 초기화 

                if(다음 버튼){ 

                  임의의 변수++; 

                  if( 임의의변수 >= 이미지-1){ 

                    임의의변수 = 0; 

                    이미지를 감싸고 있는 속성.css({left:100%}); 

                  } 

                } else {  

                  임의의변수--; 

                } 이미지를 감싸고 있는 속성.animate({ left:-100 * 임의의변수 + '%'}, function(){ 

                  if(임의의변수 < 0){ 

                    임의의변수 = 이미지수-2; 

                    이미지를 감싸고 있는 속성.css({left:-100 * 임의의변수 + '%'}); 

                  } 

                }); 

              }); 

              

              위와 같은 로직으로 가로로 움직이는 슬라이드 배너를 만들어 주면 됩니다 :)

                이미지영역.on('mouseenter', function(){ 

                  let 임의의변수 = 이미지영역.index(); 

                  이미지영역.eq(임의의변수).parents('.배경영역').css({  

                  backgroundImage:`url(${경로+이미지파일명.확장자})`}); 

                }); 

                

                일정 영역에 스크롤이 도달하면 효과를 주는 스크롤 랜딩 기법 

                mousewheel 메서드를 활용하여 효과를 줄 영역의 높이값을 

                계산하여 위와 같이 짜주면 간단하게 활용 가능합니다 :) 

                

            
          

            이미지영역.on('mouseenter', function(){ 

              let 변수 = 이미지영역.index();  

              이미지영역.css({ border:'3px solid #d68e0f'});  

              이미지영역.siblings().css({border:0});  

              텍스트영역.children('span').text(텍스트[변수]);  

              배경이미지영역.css({ backgroundImage:`url(${경로+파일명[변수]})`,  

              backgroundSize:'cover', backgroundPosition:'50% 50%',  

              backgroundRepeat:'no-repeat' })  

            });  

            

            이미지를 가지고 있는 영역에 마우스를 갖다대면 해당 이미지와  

            동일한 이미지로 뒷배경이랑 광고 문구가 바뀌는 로직입니다 :)
